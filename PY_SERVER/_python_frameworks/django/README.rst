Django Setup
============

Recommended Tutorial::

    `DjangoGirls Tutorial <https://tutorial.djangogirls.org/en/>`_

Requirements
    * Django 1.11
    * Jinja2

Step 1
------

Start project::

    django-admin.py startproject <project_name>
    cd <project_name>

Step 2
------

Create a django app::

    django-admin.py startapp <app_name>

Step 3
------

Start the django server::

    ./manage.py runserver
